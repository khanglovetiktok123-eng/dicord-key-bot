# FULL KEY SYSTEM

## Chức năng

- Web tạo key
- Discord bot quản lí key
- API check key
- Web + Bot dùng chung database

## Cài đặt

```bash
npm install
```

## Chạy web/API

```bash
node server.js
```

## Tạo slash commands

```bash
node deploy-commands.js
```

## Chạy bot

```bash
node bot.js
```

## Mở web

http://localhost:3000

## Thay thông tin

Trong:
- bot.js
- deploy-commands.js

Thay:
- PUT_BOT_TOKEN
- PUT_CLIENT_ID
